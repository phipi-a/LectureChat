# LectureChat

## Requirements

- Node.js
- npm

## Installation

```bash
cd webapp
npm install
```

## Development

```bash
cd webapp
npm run dev
```

## Info

before running the app, make sure to create a .env file in the ./webapp directory of the project and add the following variables:

```
NEXT_PUBLIC_SUPABASE_URL="..."
NEXT_PUBLIC_SUPABASE_KEY="..."
```
